# Chatbot Assignment

## Running the project
1. Clone the repo

```
git clone git@github.com:rabinghimire1999/chatbot-assignment.git
```
2. Create .env file

```
cd chat
```
```
touch .env
```

3. Enter openai key inside the .env file
OPENAI_API_KEY=<yOUR OPENAI KEY>

4. Run the project
```
streamlit run app.py
```
